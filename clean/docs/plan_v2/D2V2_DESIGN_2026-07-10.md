### 0b. 量化地板與擺動(g1_offaxis,設計約束來源)

S2 側向數據反推:該幾何(基距 ~0.48 m)下 ±0.15 m 側移的斜邊增量僅 ~0.9 cm ≈ **0.5 樣本 < 整數量化(1 樣本 = 1.77 cm)**;且觀測 peak 有 ±1.5 樣本的對稱非單調擺動(22–25)。設計後果:

1. 單次量測的有效距離誤差預算 δr ≈ 1–3.5 cm(量化 + 擺動);
2. y 估計誤差 δy ≈ δr × h / b_eff(h=水平距離,b_eff=有效基線)→ **基線必須 ≥0.3 m、視點必須 ≥5 個**(不同距離的視點使量化誤差去相關,最小平方平均化);
3. 誤差預算:目標 y ∈ ±0.15 均勻(σ=8.7 cm),r≥0.9 需 δy ≤ ~4.2 cm——以 δr=1.3 cm、b=0.3、5 視點估 δy ≈ 2–3 cm **可過**;以 δr=3.5 cm(最壞擺動)估 δy ≈ 6 cm **不過**。**真值由 g2 探針裁定,兩種結局都預先寫好**(§3 止損)。

## 1. 演算法(純幾何,離線可重算)

視點 k=1..K(K=5):感測器停於 (x_k, y_k),量 peak → 校正 → 3D 距離 r_k → 水平距離 h_k = √(r_k²−0.19²)。
目標 (x, y) 為下列最小平方解(圓交會,Gauss-Newton 或線性化兩兩相減皆可,純 stdlib):

```
minimize Σ_k [ √((x−x_k)² + (y−y_k)²) − h_k ]²
```

視點佈局(L 形,兼顧 y 基線與 x 抖動):
```
(0.60, −0.15) (0.60, −0.075) (0.60, 0) (0.60, +0.075) (0.60, +0.15)   ← 主基線 0.3 m
可選補強:(0.70, 0)(x 抖動,去相關量化)
```
每視點姿態:感測器仍水平朝向 +x(**不轉向**——D0.5 只驗證過平移;轉向是未驗證操作,不引入)。目標在 ±0.15 m 側偏、0.4–0.7 m 距離時,方位角 ≤ 21° < 視錐 45°,在 S2 側向掃描已證可測的範圍內。

## 3. D2 正式三臂(g2 過閘後)

- 場景 = D3 場景;目標 (x,y) 隨機:x∈[1.00,1.20]、y∈[−0.15,+0.15],n=30/臂,同 seed 配對。
- **closed**:5 視點定位 → 沿估計方位 2-DOF 步進接近(每步 1D 測距照舊,d̂≤0.35 停)→ 停止。
- **blind**:量測管線照跑、估距強制 +inf → 定位無解 → 沿 y=0 直走到護欄(x=0.95)停。
- **open**:零量測,直走到名義點 (1.10−0.35, 0) 停。
- 夾取:**不入主判準**(2D 對位容差 ±1–1.5 cm 超出 δy 預算;grasp 為可選附加 run,如實報)。
- 每 episode 記錄:視點量測全套(peak/波形)、(x̂,ŷ)、逐步位置、停止 2D 誤差、姿態/位姿稽核。

**預註冊判準(寫進 runner header,先於執行)**:
```
d2_loc_y_tracking   : closed r(ŷ, y_true) ≥ 0.9        ← 本實驗的新宣稱(演算法恢復側向)
d2_loc_x_tracking   : closed r(x̂, x_true) ≥ 0.95

```
