## 關鍵發現

1. **感測包絡的主宰因子是「指向」,不是「干擾」**:水平指向(pitch=0)時,桌+手臂俱在,0.10 m 目標在 0.3–0.8 m 全部可偵測(SNR ≥ 67)。俯仰 60° 幾乎全滅(立方體鏡面反射偏離接收方向)。
2. **感測器後方的手臂貢獻精確為零**:Block C 與 Block D 同幾何格點 SNR 逐位相同(148.36/94.58/67.67)——確定性模擬下,後方 clutter 對前向路徑零影響。
3. **尺寸下限**:0.04 m 目標到 1.2 m 失效;0.10 m 目標全測距段(0.15–1.2 m)可用;0.20 m 目標 SNR 最高(216–311)。
4. **與舊 grasp 場景失敗的歸因閉合**:舊場景「扳手不可見」的成因不是手臂+桌面本身,而是**感測器裝在腕上、埋在夾爪 mesh 聲影中的擺位**。V2 地圖證明:只要感測器有乾淨的前向視野,同等干擾下目標清晰可測 → 閉環在物理上可行,Stage 2 的核心設計約束 = 感測器擺位。
5. **量測方法學發現**(進論文邊界章):
   - WPM 場景建立後有數十幀確定性爬升暫態;逐幀能量恆跳動 >0.1%(輪替信號結構),僅多幀平均穩定 → runner 用 settle=40 + 事後漂移稽核。
   - 特定幾何(0.8 m/0.20 m)存在**持續性慢震盪**(drift 8→16% 隨 settle 增加),但 SNR 分母已含漂移,SNR≫門檻時「可偵測」結論仍安全;分析器規則:漂移 + SNR ≤ 3×門檻才判 INVALID。

## 對 Stage 2 的輸入

- **場景選項裁決建議**:S1 顯示不需要墊高目標或合作目標——**水平前向視野的感測器擺位**(選項空間中最接近 (a) 的變體)即可,目標 ≥ 0.10 m、工作距離 0.15–1.2 m、俯仰 ≤ 30°。最終拍板仍留給用戶/指導教授,但數據強烈支持「感測器前置、避開夾爪聲影」的構型。
