# 論文關鍵文獻 PDF 包（口試／深入閱讀用）

## 原本放在哪裡？

**本專案 git 裡原本沒有這些論文的 PDF。**
只有：
- 文內引用與參考文獻條目（`thesis/build_chapter2_docx.py`、`thesis/rebuild_thesis_v2.py`）
- 精簡對照表：`docs/plan_v2/METHOD_LITERATURE_MAP.md`
- **詳版（逐步公式 + 論文原文）：** `docs/plan_v2/EXPERIMENT_MATH_LITERATURE_GROUNDING.md`（2026-07-15）
- 筆記式文獻搜尋：`thesis/paper_rewriting_output/`

本資料夾的 PDF 主要在 **2026-07-13** 自 OA 來源下載；**2026-07-15** 補入 Nosek 2018（OSF 開放稿）。
