## 誠實記錄

1. **d3_posture_clean 判 False 的機制**:3 個 invalid 全是目標 x≈1.19(走廊最遠端)的**升舉 IK 失敗**——手臂在最大伸展處(過頭抓取半徑 ≈1.28 m,UR10e 極限 1.30)抬升 0.10 m 無解。approach 階段的姿態/感測器位姿稽核 **0 違規/130 步**,量測有效性無虞;失敗模式是可達性包絡,設計時已算出 1.20 是臨界值(runner rev9 註解)。修正方向:走廊上限縮至 1.18 或升舉高度降至 0.06 m。**判準按字面如實判 False,不做事後放寬**;其餘三條判準的數據不含這 3 個 episode 的污染(它們的對位量測仍有效且已入統計——排除它們反而會虛增對位率)。
2. **g3 scaffold 的升舉 3/10**:g3 用單點偏移掃描,±0.02 恰在捕捉窗邊界外;正式臂的連續誤差分佈下實際咬合率更高(23/30)。g3 的價值在量出窗寬與驗證機動,不在預測成功率。
3. **weld 的科學邊界**:本模擬器 2F-85 的摩擦保持在 D3 姿態鏈下不可用(E2 原物復刻亦失敗;完整歸因鏈=24 輪對照實驗,d3/notes.md)。weld 觸發是物理接觸訊號,對照組區辨力保留——blind 的 4 次升舉全部來自運氣對位後的真實咬合,非機制漏洞。
4. **與 D1.5 的關係**:D1.5 證了「聲學停止」(停止誤差 2.8 cm);D3 把鏈路延伸到「停止後的終端推進 + 夾取對位」(對位 RMSE 1.9 cm——推進以停止時刻聲學估距 dead-reckoning,0.32 m 內無聲學宣稱,D-6)。
