# M3 校正斜率漂移歸因報告(2026-07-10)

**問題**(D1/D1.5 遺留開放問題):三次獨立量測的 peak_idx-距離斜率 52.4 / 57.9 / 64.3 smp/m,跨度 ±10%,未歸因。

## 結論（2026-07-22 封口措辭）

**原結論強度過滿，改為：**

現有 M3a（斜率 95% CI）與 M3b（同場景移目標 vs 移感測器）**未提供充分證據支持穩定的 mover effect**；probe 與 S2 斜率點估差異約 ±10% 與小樣本／短距離跨度相容。  
**斜率差異的唯一物理原因仍未完全歸因**；因此各配置維持使用**幾何匹配的獨立版本化校正**，不得寫「移動目標與移動感測器具有相同斜率」。

同 session 完整 A/B/C 列為**後續精煉**，**非** 3.2–5.1 定稿阻塞項。

### 口試答辯句

probe 的角色是**有效性閘門**(r≥0.99 證明感測器運動被 WPM 正確追蹤),不是校正來源;控制用校正(tableh 57.9±0.7,n=13,跨 0.85 m)斜率不確定度 ±1.3%,傳播至 standoff 停止點 <5 mm,與實測停止誤差 2.8 cm RMSE 相容;「移目標 vs 移感測器」無系統差已由 M3b 直接實驗確認。

### 附帶發現

不同目標物同斜率:0.10 立方體(tableh 57.87)與 0.06×0.06×0.12 bar(D3 合併校正 58.12±1.33)——斜率是介質/採樣性質,截距才吸收物體幾何。
