> **保持 D2 圓交會為正典。**  
> 經驗距離 \(\hat r\) 在正式資料上貼近**感測器原點單站距離** \(R\)，而非雙站 \(L/2\)；圓模型與 OLS 觀測定義**匹配**。

## 1. 問題怎麼提才對

外部／P0 遺留問題常寫成：

> 兩個掛載相距 0.10 m → 路徑是橢圓 → 圓交會是否錯？

完整答案必須拆兩層：

| 層 | 問題 | 本報告答案 |
|----|------|------------|
| **物理路徑** | 若峰對應 TX→目標→RX 總長 \(L\) | 軸向時 \(L/2 - R \approx -s/2 \approx -5\) cm |
| **觀測定義** | V2 的 \(\hat r\) 是什麼？ | OLS：peak → **真值單站距離**（感測器原點到目標） |
| **交會模型** | 應用圓還是橢圓？ | 對 \(\hat r\approx R\) 用**圓**；對原始 \(L\) 才用橢圓 |

D2 runner 做的是：

```text
peak → OLS → r3d_est ≈ R
h = √(r3d² − Δz²)
minimize Σ ( ‖(x,y)−(x_k,y_k)‖ − h_k )²   # 圓
```

---

## 2. 掛載幾何假設

與 `rtx_acoustic_factory` 一致：

- m001 @ 感測器原點 \(s\)
- m002 @ \(s + (0.10, 0, 0)\)（感測器前方 +X，D2 水平前視）
- 雙站路徑 \(L = |p-s| + |p - (s+(0.10,0,0))|\)
- 單站 \(R = |p-s|\)

軸向遠場：\(L = R + (R - 0.10) = 2R - 0.10 \Rightarrow L/2 - R = -0.05\) m。

---

## 3. 合成網格（D2 目標盒 × 五視點）

目標 \(x\in[1.0,1.2]\)，\(y\in[-0.15,0.15]\)；視點 \(x=0.60\)，\(y\in\{\pm0.15,\pm0.075,0\}\)。

| 量 | 值 |
|----|-----|
| \(L/2 - R\) 均值 | **−4.36 cm** |
| \(L/2 - R\) \|max\| | **4.70 cm** |
| 完美 \(R\) + **圓** 定位 RMSE | **≈ 0** |
| 誤用 \(L/2\) 當圓半徑 RMSE | **4.95 cm** |
| 真 \(L\) + **橢圓** 定位 RMSE | **4.42 cm**（數值/平面化殘差；匹配模型應近 0，此處受 3D→2D 水平化與種子影響，仍遠小於「錯用觀測」時的錯誤敘事） |

解讀重點不在橢圓 RMSE 小數點，而在：

- **匹配的單站+圓 → 誤差可壓到數值零**  
- **把 \(L/2\) 硬塞進圓 → ~5 cm 系統誤差**（與掛載半距同量級）

## 4. 正式三臂 closed 資料（n=30 ep × 5 視點 = 150）

| 比較 | bias | RMSE | 結論 |
|------|-----:|-----:|------|
| \(\hat r - R_{\mathrm{mono}}\) | **+0.12 cm** | **1.46 cm** | 貼單站 |
| \(\hat r - L/2\) | **+4.48 cm** | **4.71 cm** | 系統性偏大 ~半間距 |
| 平均絕對誤差 | vs \(R\): 1.20 cm | vs \(L/2\): 4.48 cm | **明確偏好 \(R\)** |

定位（相對真值）：

| 方法 | 2D 定位 RMSE |
|------|-------------:|
| 實驗紀錄 \(x̂,ŷ\)（圓+估計距） | **3.54 cm** |
| 離線重跑（同 \(\hat r\)+圓） | 3.53 cm |
| Oracle \(R\) + 圓 | ≈ 0 |
| Oracle \(L/2\) + 圓（錯模型） | 4.94 cm |
| Oracle \(L\) + 橢圓 | 4.46 cm |

實驗 ~3.5 cm 側向誤差來自**測距噪聲／校正／姿態**，不是「該用橢圓卻用了圓」的主因；若觀測真是 \(L/2\) 卻用圓，會再疊 ~5 cm 級系統項——但資料顯示觀測**不是** \(L/2\)。

## 6. 論文／口試怎麼講

**可寫：**

- D2 使用與一維閉環相同的經驗距離 \(\hat r=(k-b)/a\)，該量在腕載幾何下對齊**感測器原點至目標的單站距離**；多視點最小平方因此採用**圓**約束，與觀測定義一致。  
- 配置上存在 0.10 m 雙掛載；純幾何雙站路徑滿足 \(L/2-R\approx -s/2\)。本文**不**以未校正 ToF 直接當 \(L\)，故不改用橢圓交會為正典。  
- 側向 RMSE ~3.3 cm（正式報告）主因是測距量化／殘差與基線幾何，而非圓／橢圓選錯。

**不可寫：**

- 「已證明物理傳播為單站、雙站不存在」  
- 「橢圓模型已驗證並優於圓」  
- 「0.10 m 分離可忽略到無需在 limitation 提及」
