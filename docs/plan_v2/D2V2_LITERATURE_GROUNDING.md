# D2v2 與 V2 流程的文獻地基(2026-07-10 檢索)

**動機**:執行 rxGroup 閘門/D2v2 前,確認設計元素皆有既有文獻支撐(非憑空生成)。

## 新穎性空窗(= 論文貢獻主張的文獻依據)

針對「ultrasonic closed-loop grasping / acoustic servoing for manipulation」的檢索**顯著稀疏**——結果被視覺伺服與觸覺夾取佔據,無超聲閉環夾取的直接先例。這是可寫進 CH1/CH2 的 gap statement:超聲在**行動機器人定位/導航**文獻豐富,在**機械臂閉環操作**幾乎空白;本研究以模擬可行性研究填補此空窗。

## 對「流程是否紮實」的裁定

1. 方法組件全部有譜系(上表);
2. 實驗紀律(預註冊判準、配對對照、消融)是實驗科學標準配備,消融部分有機器人領域直接文獻;
3. 新穎點落在「組合與場景」——正是碩論該有新穎性的位置,而非發明未經驗證的新方法。
